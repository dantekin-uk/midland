# Assets Folder

This folder contains static assets for the Midland Geosystems website.

## Hero Video

**Expected file:** `hero.m4a` (note: should be .mp4 format for web compatibility)

### Video Requirements:
- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 (Full HD) or higher
- **Duration:** 10-30 seconds (looped)
- **File size:** Under 10MB for optimal loading
- **Content:** Surveying/infrastructure projects in Kenya

### Suggested Video Content:
1. Opening shot: Aerial view of large Kenyan infrastructure project
2. Middle shot: Surveyor using modern equipment (total station, GPS)
3. Final shot: UAV/drone taking off for aerial surveying

- ✅ Assets folder created

### To add your video:
1. Place your `hero.m4a` file in this folder
2. Rename to `hero.mp4` for better web compatibility
3. Update the path in `HeroSection.jsx` if needed

The website will automatically use the video when it's available, or show a beautiful gradient fallback until then.
